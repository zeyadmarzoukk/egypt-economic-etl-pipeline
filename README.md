# Egypt Economic ETL Pipeline

An automated data pipeline that extracts Egypt's key economic indicators from the World Bank API, transforms and cleans the data, and loads it into a SQLite database for analysis.

## What it does
- Extracts GDP, inflation, and unemployment data for Egypt (30 years)
- Cleans and transforms the data using Pandas
- Loads into a SQLite database
- Data sourced from the World Bank public API

## Tech Stack
- Python 3
- Pandas
- Requests
- SQLite3
- World Bank API
